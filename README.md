# myBallsUT4_2
Inicialización Unity


RESUMEN

Crea un sencillo juego de una bola rodante que te enseñará muchos de los principios para trabajar con Unity.

No se requiere la descarga de ningún recurso ("asset") para este proyecto.

OBJETIVO

Enseña muchos de los principios de cómo trabajar con Objetos de Juego ("Game Objects"), Componentes ("Components"), Prefabs, Física ("Physics") y Scripting (programación de scripts).
